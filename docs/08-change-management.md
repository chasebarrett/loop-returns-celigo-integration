# Change Management

## Stakeholders impacted
- Customer Service (visibility + customer comms)
- Warehouse (receiving + inspection ownership shift)
- Product Services (reduced inspection bandwidth load)
- Finance (credit memo + audit trail)
- Ecommerce/Systems (integration monitoring + exception handling)

## Adoption strategy
- Phase 1 ran in parallel with legacy processes long enough to stabilize portal behavior
- Training focused on:
  - How returns appear in Loop Admin
  - When and how to create Item Receipts in NetSuite
  - What “good” vs “exception” returns look like
  - Escalation paths when data doesn’t sync as expected

## Documentation
- SOPs: Receiving + inspection + item receipt creation
- Exception playbooks: what to do when a record fails or mismatches
- Customer messaging templates (portal expectations, timelines)
